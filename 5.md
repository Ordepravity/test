# 代维人员考勤与巡检统计技能

## 技能简介
该技能用于汇总代维人员的签到记录、巡检任务完成情况和异常事件，生成考勤与巡检统计结果，并支持报表导出与周期分析。

## 实验目标映射
1. 掌握多数据源整合方法。
2. 构建考勤统计模型。
3. 实现巡检任务完成率计算。
4. 完成多维度报表生成。
5. 支持日期范围筛选和导出。

## 核心处理流程
1. 读取配置文件与基础数据。
2. 加载代维人员列表、签到记录、巡检任务数据。
3. 根据统计周期过滤有效数据。
4. 按人员聚合签到记录，统计应到、实到、迟到、早退、缺勤。
5. 计算出勤率与准时率。
6. 按人员和任务类型聚合巡检任务。
7. 统计计划任务数、完成任务数、待处理任务数、取消任务数。
8. 计算总体完成率与各任务类型完成率。
9. 识别完成率低于阈值的人员。
10. 输出控制台汇总信息并生成 Excel 报表。

## 规则说明
### 考勤统计规则
- 使用 `work_start_time` 作为上班时间基准。
- 使用 `work_end_time` 作为下班时间基准。
- 签到晚于上班时间且超过 `late_threshold_minutes` 记为迟到。
- 签退早于下班时间且超过 `early_leave_threshold_minutes` 记为早退。
- 无有效签到记录的日期计为缺勤。
- 出勤率 = `actual_days / total_days * 100%`
- 准时率 = `(actual_days - late_count) / actual_days * 100%`

### 巡检统计规则
- 仅统计处于查询周期内的任务。
- 巡检完成率 = `completed_tasks / total_tasks * 100%`
- 单任务完成率 = `completed_items / items * 100%`
- 若人员完成率低于 `completion_threshold * 100`，则纳入待关注人员列表。

## 异常处理建议
- 当签到记录缺失时，标记为缺勤并输出说明。
- 当任务字段为空时，按默认规则计算完成率。
- 当无统计数据时，返回空结果并给出提示。
- 当输出目录不存在时，自动创建目录。
