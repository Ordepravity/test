# 基站告警智能分析技能文档

## 技能简介
该技能用于对基站告警进行智能分析，包括告警分类、严重程度评分和报告生成。

## 输入
- `alarm_id`
- `ne_type`
- `ne_name`
- `alarm_title`
- `alarm_content`
- `occur_time`
- `clear_time`

## 输出
根据输入生成告警分析报告。

## 核心处理流程
1. 根据规则进行告警分类。
2. 计算严重程度和优先级评分。
3. 提取告警位置。
4. 生成总结报告与推荐措施。
5. 确定优先级最高的告警。

- 严重性分级：

  - 主要 (Major)

- 优先级评分逻辑：
  - 源权重
  - 未清除时间加权

## 依赖数据/接口

## 执行步骤
1. 收集告警信息。
2. 应用分类规则。
3. 计算严重程度与优先级。
4. 生成报告。
